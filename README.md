# 🔬 [물리학 II 탐구 활동] 음성 신호의 분석과 푸리에 변환

## 1. 프로젝트 개요
이 프로그램은 우리가 일상에서 듣는 '소리(음성 신호)'를 수학적, 물리적으로 분석하여 시각화해주는 STFT(Short-Time Fourier Transform) 시각화 도구입니다. 복잡한 파동인 소리를 단순한 사인파들의 합으로 분해하는 과정을 단계별로 체험할 수 있습니다.

---

## 2. 주요 물리 및 수학적 원리 (고교 물리학 II 수준)

### 가. 파동의 중첩과 분해
* 모든 복잡한 소리 파동은 진동수와 진폭이 다른 여러 개의 단순한 파동(사인파, 코사인파)들이 겹쳐진 형태입니다.
* 푸리에 변환(Fourier Transform)은 이렇게 겹쳐진 파동에서 "어떤 진동수의 소리가 얼마나 섞여 있는지"를 찾아내는 수학적 기법입니다.

### 나. 시간-진동수 불확정성
* 소리는 시간에 따라 계속 변화합니다. 전체 소리를 한꺼번에 분석하면 "언제" 그 소리가 났는지 알 수 없습니다.
* 이를 해결하기 위해 소리를 아주 짧은 시간(Frame) 단위로 나누어 분석하는데, 이를 '단시간 푸리에 변환(STFT)'이라고 합니다.

### 다. 창 함수 (Window Function)
* 소리를 짧게 자를 때 양 끝부분에서 갑작스러운 불연속성이 생겨 오차가 발생할 수 있습니다.
* 이를 부드럽게 만들어주기 위해 '한 창(Hann Window)'이라는 함수를 곱하여 분석의 정확도를 높입니다.

---

## 3. 프로그램 시각화 단계

1.  [Step 1] 소리 데이터 추출 (Time Domain)
    * 업로드한 음원 파일에서 시간에 따른 공기 압력의 변화(진폭) 데이터를 추출합니다.
2.  [Step 2] 구간 나누기 및 창 함수 적용
    * 전체 데이터를 짧은 구간(2048 샘플)으로 나누고, 해닝 창을 적용하여 분석 준비를 합니다.
3.  [Step 3] 고속 푸리에 변환 (FFT) 실행
    * FFT 알고리즘을 통해 '시간축' 데이터를 '진동수(주파수)축' 데이터로 변환합니다.
4.  [Step 4] 로그 스케일 시각화
    * 인간의 청각 특성을 반영하여 주파수를 로그(Log) 단위로, 세기를 데시벨(dB)로 변환하여 보여줍니다.
5.  [Step 5] 스펙트로그램(Spectrogram) 완성
    * 시간별 주파수 데이터를 쌓아 올려 '소리의 지도'를 완성합니다.

---

## 4. 탐구 가치
이 도구를 통해 물리학 II에서 배우는 '파동의 중첩' 원리가 현대 디지털 신호 처리(음성 인식, AI 음악 분석 등)에 어떻게 필수적으로 사용되는지 깊이 있게 이해할 수 있습니다.
